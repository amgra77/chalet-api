# chalet-api
API to manage a chalet. Testing [Fastify](https://www.fastify.io/) Javascript framework
* Fastify
* Fastify-Swagger
* Fastify-JWT
* Fastify-Mongo

# Run
First copy `.env.sample` into `.env` and modify to your settings (db credentials, etc...)

```shell
npm install
npm run start
```
